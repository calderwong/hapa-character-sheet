## Sharpened Kernel

**Index the living archive -> distill the useful pattern -> encode it as protocol -> make it runnable through UI/API/CLI -> make it beautiful enough that humans want to enter it -> keep the proof visible so agents can safely continue it.**

That pattern appears across Hapa Protocol turns, Character Sheet work, node/app requests, media prototypes, and the repeated demand that agents be able to start, authenticate, test, and explain their own work.

## Sharpened Voice Model

Calder's command style has several layers:

1. **Map request:** "give me as complete a summary..." / "index, distill..." / "review..."
2. **Protocol request:** "apply to Hapa protocol" / "per protocol..." / "up to Hapa protocol standards"
3. **Execution request:** "make it so" / "run it for me" / "start/authenticate/test"
4. **Refinement request:** "make it comprehensive" / "make it interesting" / "sharpen it"
5. **Trust check:** "you asked if I wanted the full list..." / "assumed you could do it"

Agent implication: respond by doing the loop, not by narrating the loop forever.
